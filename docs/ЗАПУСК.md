# JobTrack — краткий запуск

Полная инструкция, переменные, troubleshooting и reseed — в корневом **`README.md`**. Здесь — только чеклист.

## Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows/macOS) или Docker Engine + Compose (Linux)
- В терминале: `docker version` показывает **Client** и **Server**

## Команды (из корня репозитория)

```powershell
copy .env.example .env
docker compose up -d --build
```

Открыть в браузере: **http://localhost**  
API: **http://localhost:8000/docs**

**Демо:** `demo@example.com` / `DemoPass123!`

## Остановка

```powershell
docker compose down
```

## Повторная заливка демо-данных

После обновления кода сначала пересоберите backend, затем:

```powershell
docker compose build backend
docker compose up -d
docker compose exec backend python -m scripts.reseed_demo
```

Подробности — в **README.md** (раздел «Повторная заливка демо-данных»).
